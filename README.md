# advanced-js-dz

Решения в папке src


Для запуска вызывайте:
```
node .\src\allKeysAndSymbols.js
node .\src\in.js
node .\src\asyncExecutor.js
```

У меня Node v12.13.0

Для функции allKeysAndSymbols написаны тесты, запускаются (не в песочнице) после установки mocha+chai через
```
npm install
npm test
```
